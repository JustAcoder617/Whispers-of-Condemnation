## 🔧 Componentes Centrais

### 🔐 cweb.h – O Sistema Nervoso do Motor

> "O cweb.h foi desenvolvido para ser o **sistema nervoso** do motor. Ele não apenas envia dados; ele gerencia a segurança através de **descriptografia em tempo de execução** utilizando a robustez da `libcurl`, **incluindo verificação para saber se há internet ou não.**"

#### Funcionalidade Principal

```c
static inline char* descriptografar(char *dados, int tamanho, int chave)
```

#### O que oferece:

| Feature | Descrição |
|---------|-----------|
| 🔐 **Segurança Ofuscada** | Proteção de configurações via operação XOR |
| 📝 **JSON Dinâmico** | Formatação automática para integração com Discord |
| 📊 **Telemetria de Erros** | Envio automático de logs e feedbacks |

---

### 🧠 Estrutura "Expressao" – A Grande Estrela

A função `Expressao` centraliza **todo o processamento matemático** em uma única estrutura robusta, eliminando a necessidade de ifes e elses espalhados pelo código.

#### Capacidades:

```
📐 Aritmética Básica e Avançada (Soma, Subtração, Multiplicação, Divisão)
🔋 Potenciação Direta
🔄 Conversão Inteligente (Fração → Decimal no Motor)
```
